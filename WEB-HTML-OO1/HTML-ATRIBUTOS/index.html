    • ATRIBUTOS DEL LENGUAJE HTML

    ¿Qué son los atributos?
    Los atributos en el lenguaje HTML proporcionan información adicional a los elementos

      ¿Que pueden ser?
    • Todos los elementos de HTML pueden tener atributos
    • Los atributos proporcionan información adicional sobre los elementos
    • Lsos atributos siempre se especifian en la etiqueta de incio.
    • Los atributos generalmente vienen en pares de nombre + el valor como, nombre="valor".

      Uno de los atributos más usados, es el href.

    • El href especifica la URL de la página que lo va el enlace.
    • Normalmente va acompañado de la etiqueta < a>.

      • El atributo src.
    • Cuando pones, usas la etiqueta de la imagen, que es < img> , el atributo src sirve para especificar la imagen que se va a mostrar.,
    • Así quedaria en un ejemplo, < img src="imagen.jpg">.

      Los atributos de alto y ancho


    • La etiqueta de la imagen, < img> deben de contener atributos de width y height, las cuales definen el largo y la altura de la imagen.

    • El atributo alt.

      El alt atributo que lo requiere la etiqueta < img> especifica un texto alternativo para la imagen, si la imagen por algún motivo no se puede mostrar. Esto puede deberse a una conexión lenta, a un error en el srcatributo, o si el usuario usa un lector de pantalla.

    • El Atributo de estilo

      El Style en su forma de atributo, < style> , se usa para agregar estilo en linea, es decir en la misma linea del elemento.
      Ejemplo:

      < p style="color-red">TODO ESTE TEXTO ESTARIA EN ROJO< /p>

    • El atributo lang

      El atributo < lang> , se incluye dentro del html, y se usa para epecificar el idioma de la pagina.
      Como por ejemplo: < html lang="es">, para el caso de la pagina ser en español.
